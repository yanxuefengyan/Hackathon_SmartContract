# 智合同 SmartContract  
**AI双端协同平台：采购+销售合同全流程智能生成与审核系统**

> 基于文心大模型的智能合同处理平台，覆盖采购与销售双场景，实现“报价单→合同→审核→盖章”全流程自动化，支持WEB与APP双端协同。

---

## 📌 项目简介

**智合同**是一款面向企业采购与销售合同场景的智能协同平台，通过AI大模型驱动，实现从报价单到合同的智能生成、合规审核、跨端审批与电子盖章的全流程自动化。系统支持WEB端（批量处理、知识库管理）与APP端（移动上传、实时审批）双端协同，助力企业提效80%、降低合同风险。

---

## 🎯 核心功能

### 1. **采购合同模块（WEB + APP）**
- **报价单 → 合同智能生成**  
  支持上传PDF/图片报价单，OCR识别提取信息，自动生成标准化采购合同。
- **供应商合同智能审核**  
  基于采购知识库识别风险条款（如质保期、付款方式），生成审核报告。
- **跨端审批与电子盖章**  
  支持WEB/APP双端审批流程，一键触发电子签章，结果实时同步。

### 2. **销售合同模块（WEB + APP）**
- **销售报价单智能生成**  
  输入客户信息与产品明细，自动生成标准化报价单，支持自定义条款。
- **报价单 → 销售合同自动转化**  
  基于已审批报价单，补充违约责任等核心条款，生成完整销售合同。
- **移动端语音输入快速发起**  
  支持APP端语音输入生成报价单草稿，提升移动办公效率。

### 3. **双端协同适配**
- **WEB端**：批量操作、知识库管理、数据报表、文件导出
- **APP端**：拍照上传、一键审批、实时提醒、离线查看
- **数据实时同步**：合同状态、审批进度、文件版本双端一致

## 🏗️ 技术架构

### 核心技术依赖
- **基础模型**：文心大模型 4.0（文本生成、OCR、语义理解）
- **开发工具**：文心快码、Vibe Coding、移动应用开发框架
- **关键能力**：多智能体框架、MCP工具、OCR识别引擎、流程自动化、双端数据同步、电子签章API、文件管理系统

### 分层架构设计
1. **接入层**：WEB响应式界面 + APP轻量化界面 + 多模态输入
2. **核心层**：
   - 多智能体系统（调度Agent、采购/销售Agent、知识库管理Agent等）
   - MCP工具系统（多模态内容处理、流程自动化）
3. **文件管理层**：合同知识库、文件库、操作日志库、系统配置库（基于文件管理系统）

## 🚀 快速开始

### 环境要求
- Node.js ≥ 16
- Python ≥ 3.8
- 文心大模型API权限
- 电子签章系统对接（如法大大、e签宝）

### 安装与运行
​```bash
# 克隆仓库
git clone https://github.com/your-username/smart-contract-platform.git

# 进入项目目录
cd smart-contract-platform

# 安装依赖
npm install  # 或 yarn install

# 启动开发环境
npm run dev
```

### 配置说明
1. 在各服务目录下配置API密钥、文件存储路径等（参考各目录下的`.env.example`）
2. 初始化知识库：运行 `scripts/init-knowledge-base.py`
3. 初始化文件管理系统：运行 `npm run init:file-index`（在server目录下）
4. 启动服务：参考 `docs/setup.md`
